# Todo In-Memory Console App - Specification v1

## Objective
Build a Python console-based todo application that stores tasks in memory.

## Features
1. Add new task with title and description
2. View all tasks
3. Update existing tasks
4. Delete tasks by ID
5. Mark task complete/incomplete

## Data Model
Each task has:
- id (int)
- title (str)
- description (str)
- completed (bool)

## Interface
Menu-based CLI:
1. Add Task
2. View Tasks
3. Update Task
4. Delete Task
5. Toggle Complete
6. Exit

## Constraints
- No file/database storage
- Data stored in memory
- Python 3.12+ (your current Python is fine)
